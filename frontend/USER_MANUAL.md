# 클래스 플래너 사용자 매뉴얼

## 📋 개요

클래스 플래너는 학생과 수업을 효율적으로 관리할 수 있는 웹 기반 시간표 관리 시스템입니다. 직관적인 인터페이스와 강력한 기능을 통해 교육 기관의 수업 관리를 간소화합니다.

---

## 🚀 시작하기

### 📱 접속 방법

1. 웹 브라우저를 열고 애플리케이션에 접속합니다
2. 자동으로 학생 관리 페이지가 표시됩니다

### 🎨 테마 설정

- 우측 상단의 테마 토글 버튼을 클릭하여 다크/라이트 모드를 전환할 수 있습니다
- 설정은 자동으로 저장되어 다음 접속 시에도 유지됩니다

---

## 👥 학생 관리

### ➕ 학생 추가

1. **학생 이름 입력**: 좌측 패널의 입력창에 학생 이름을 입력합니다
2. **추가 버튼 클릭**: "추가" 버튼을 클릭하거나 Enter 키를 누릅니다
3. **확인**: 학생이 목록에 추가되고 입력창이 초기화됩니다

#### ⚠️ 주의사항

- **중복 이름 방지**: 이미 존재하는 학생 이름은 추가할 수 없습니다
- **빈 이름 방지**: 빈 이름이나 공백만 있는 이름은 추가할 수 없습니다
- **에러 메시지**: 문제가 발생하면 적절한 에러 메시지가 표시됩니다

### 👀 학생 목록 관리

- **학생 선택**: 학생 이름을 클릭하여 선택 상태로 변경합니다
- **선택 해제**: 선택된 학생을 다시 클릭하여 선택을 해제합니다
- **삭제**: 각 학생 옆의 삭제 버튼을 클릭하여 학생을 제거합니다

### 🎯 학생 선택의 효과

- 선택된 학생이 있으면 시간표에서 해당 학생의 수업만 표시됩니다
- 선택을 해제하면 모든 학생의 수업이 표시됩니다

---

## 📅 시간표 관리

### 🎯 시간표 보기

- **시간 범위**: 9:00부터 23:00까지 30분 단위로 표시됩니다
- **요일 표시**: 월요일부터 일요일까지 7일이 표시됩니다
- **세션 표시**: 등록된 수업이 해당 시간대에 색상별로 표시됩니다

### 🖱️ 수업 추가 (드래그 앤 드롭)

1. **학생 드래그**: 우측 플로팅 패널에서 학생을 드래그합니다
2. **시간대에 드롭**: 원하는 요일과 시간대에 드롭합니다
3. **모달 입력**: 수업 정보 입력 모달이 열립니다
4. **정보 입력**: 과목, 시간, 강의실 등을 입력합니다
5. **추가 완료**: "추가" 버튼을 클릭하여 수업을 등록합니다

### 🖱️ 수업 추가 (빈 공간 클릭)

1. **빈 공간 클릭**: 시간표의 빈 공간을 클릭합니다
2. **모달 입력**: 수업 정보 입력 모달이 열립니다
3. **정보 입력**: 학생, 과목, 시간, 강의실 등을 입력합니다
4. **추가 완료**: "추가" 버튼을 클릭하여 수업을 등록합니다

### ✏️ 수업 편집

1. **수업 클릭**: 시간표에서 편집할 수업을 클릭합니다
2. **모달 열림**: 수업 편집 모달이 열립니다
3. **정보 수정**: 시간, 강의실 등을 수정합니다
4. **저장**: "저장" 버튼을 클릭하여 변경사항을 적용합니다

### 🗑️ 수업 삭제

1. **수업 클릭**: 삭제할 수업을 클릭합니다
2. **삭제 버튼**: 편집 모달에서 "삭제" 버튼을 클릭합니다
3. **확인**: 수업이 시간표에서 제거됩니다

### 🆕 시간 검증 시스템

- **자동 검증**: 시작 시간이 종료 시간보다 늦을 수 없습니다
- **경고 메시지**: 무효한 시간 범위 입력 시 경고 메시지가 표시됩니다
- **수동 수정**: 시간은 자동으로 조정되지 않으므로 사용자가 직접 수정해야 합니다
- **실시간 피드백**: 모달에서 실시간으로 시간 유효성을 확인할 수 있습니다

### 🆕 학생 입력 개선

- **엔터 키 초기화**: 학생 이름을 입력하고 엔터를 치면 입력창이 완전히 비워집니다
- **수업 추가/편집 모달**: 두 모달 모두에서 동일하게 작동합니다
- **편의성 향상**: 연속으로 여러 학생을 추가할 때 더욱 편리합니다

### 🆕 겹치는 세션 처리

- **개별 표시**: 동일한 시간대의 세션들이 모두 개별적으로 표시됩니다
- **기존 문제 해결**: 이전에는 첫 번째 세션만 표시되어 다른 세션이 숨겨지는 문제가 있었습니다
- **시각적 개선**: 모든 세션이 적절한 높이에 배치되어 모두 볼 수 있습니다

## 🔍 플로팅 패널

### 📍 패널 위치

- **기본 위치**: 화면 우측 상단에 표시됩니다
- **드래그 이동**: 패널 헤더를 드래그하여 위치를 변경할 수 있습니다
- **위치 저장**: 변경된 위치는 자동으로 저장됩니다

### 🔍 학생 검색

1. **검색창 클릭**: 패널 상단의 검색창을 클릭합니다
2. **이름 입력**: 찾고자 하는 학생 이름을 입력합니다
3. **실시간 필터링**: 입력한 텍스트로 학생 목록이 실시간 필터링됩니다
4. **검색 결과**: 일치하는 학생만 목록에 표시됩니다

### 🎯 학생 선택

- **개별 선택**: 학생 이름을 클릭하여 선택/해제할 수 있습니다
- **필터링 효과**: 선택된 학생의 수업만 시간표에 표시됩니다

---

## 📄 PDF 다운로드

### 🖨️ PDF 생성

1. **다운로드 버튼**: 시간표 우측 상단의 PDF 다운로드 버튼을 클릭합니다
2. **자동 생성**: 현재 표시된 시간표가 PDF로 변환됩니다
3. **파일 저장**: 브라우저의 다운로드 폴더에 저장됩니다

### 📊 PDF 내용

- **선택된 학생**: 학생이 선택되어 있으면 해당 학생의 시간표만 포함됩니다
- **전체 시간표**: 학생이 선택되어 있지 않으면 모든 학생의 시간표가 포함됩니다
- **A4 가로 방향**: 프린트에 최적화된 가로 방향 레이아웃입니다
- **고품질**: 인쇄용 고품질로 생성됩니다

### 🆕 다중 학생 그룹 수업

- **여러 학생 추가**: 한 번에 여러 학생을 그룹 수업에 추가할 수 있습니다
- **개별 관리**: 각 학생을 개별적으로 추가/제거할 수 있습니다
- **자동 enrollment 생성**: 수업에 추가된 학생의 enrollment가 자동으로 생성됩니다
- **그룹 관리**: 그룹 수업의 학생 목록을 실시간으로 확인할 수 있습니다

### 🆕 세션 범위 기반 PDF

- **자동 범위**: 가장 빠른 시작 시간부터 가장 늦은 종료 시간까지만 표시됩니다
- **공간 효율성**: 불필요한 빈 시간대는 제외되어 공간을 효율적으로 사용합니다
- **가독성 향상**: 실제 수업이 있는 시간대만 표시되어 가독성이 향상됩니다

- **에러 방지**: 잘못된 데이터 입력을 방지하는 시스템이 구축되어 있습니다

## 🚨 문제 해결

### ❓ 자주 묻는 질문

#### Q: 학생을 추가할 수 없습니다

**A**: 다음을 확인해주세요:

- 학생 이름이 비어있지 않은지 확인
- 이미 존재하는 학생 이름인지 확인
- 브라우저의 저장 공간이 충분한지 확인

#### Q: 수업이 시간표에 표시되지 않습니다

**A**: 다음을 확인해주세요:

- 학생이 선택되어 있는지 확인 (선택된 학생의 수업만 표시됨)
- 올바른 시간대에 수업을 추가했는지 확인
- 브라우저를 새로고침하여 데이터를 다시 로드

#### Q: PDF 다운로드가 작동하지 않습니다

**A**: 다음을 확인해주세요:

- 브라우저의 팝업 차단 설정 확인
- 다운로드 폴더의 권한 확인
- 인터넷 연결 상태 확인

#### Q: 시간표가 제대로 표시되지 않습니다

**A**: 다음을 확인해주세요:

- 브라우저를 최신 버전으로 업데이트
- 캐시를 삭제하고 페이지 새로고침
- 다른 브라우저로 시도

### 🔧 기술적 문제

#### 브라우저 호환성

- **권장 브라우저**: Chrome, Firefox, Safari, Edge 최신 버전
- **최소 요구사항**: ES6 지원 브라우저

#### 성능 최적화

- **대용량 데이터**: 많은 학생과 수업이 있는 경우 로딩 시간이 길어질 수 있습니다
- **메모리 사용**: 브라우저의 메모리 사용량을 모니터링하세요
