# 클래스 플래너 사용자 매뉴얼

## 📋 개요

클래스 플래너는 학원이나 개인 과외 선생님이 학생들의 수업 일정을 효율적으로 관리할 수 있도록 도와주는 웹 애플리케이션입니다.

---

## 🚀 주요 기능

### 📚 학생 관리

- **학생 추가**: 새로운 학생을 쉽게 추가할 수 있습니다
- **학생 삭제**: 더 이상 수업을 받지 않는 학생을 삭제할 수 있습니다
- **학생 선택**: 특정 학생의 수업만 필터링하여 볼 수 있습니다
- **중복 방지**: 동일한 이름의 학생은 추가할 수 없습니다

### 📅 시간표 관리

- **시각적 시간표**: 9:00부터 23:00까지 30분 단위로 시간표를 표시합니다
- **드래그 앤 드롭**: 학생을 드래그하여 원하는 시간대에 수업을 추가할 수 있습니다
- **수업 편집**: 기존 수업의 시간, 과목, 학생을 수정할 수 있습니다
- **수업 삭제**: 불필요한 수업을 삭제할 수 있습니다

### 📖 과목 관리

- **기본 과목**: 초등수학, 중등수학, 중등영어, 중등국어, 중등과학, 중등사회, 고등수학, 고등영어, 고등국어가 자동으로 생성됩니다
- **과목별 색상**: 각 과목마다 고유한 색상이 지정되어 시각적으로 구분됩니다

### 📊 데이터 관리

- **자동 저장**: 모든 데이터는 브라우저에 자동으로 저장됩니다
- **데이터 복원**: 페이지를 새로고침해도 데이터가 유지됩니다
- **PDF 다운로드**: 현재 시간표를 PDF로 다운로드할 수 있습니다
- **패널 위치 저장**: 수강생 리스트 패널의 위치가 자동으로 저장됩니다

### 🎨 사용자 인터페이스

- **플로팅 패널**: 수강생 리스트가 드래그 가능한 플로팅 패널로 표시됩니다
- **직관적 UX**: 마우스 커서 변경으로 드래그 가능함을 알려줍니다
- **반응형 디자인**: 다양한 화면 크기에 대응합니다
- **다크/라이트 테마**: 사용자 선호에 따라 테마를 변경할 수 있습니다

## 🎯 사용 방법

### 1. 학생 관리 페이지 (`/students`)

#### 학생 추가하기

1. 좌측 학생 목록 섹션에서 "학생 이름" 입력 필드에 이름을 입력합니다
2. "추가" 버튼을 클릭하거나 Enter 키를 누릅니다
3. 추가된 학생이 목록에 표시됩니다

#### 학생 선택하기

1. 학생 목록에서 원하는 학생을 클릭합니다
2. 선택된 학생은 하단에 "선택된 학생: [이름]"으로 표시됩니다
3. Schedule 페이지에서 해당 학생의 수업만 필터링됩니다

#### 학생 삭제하기

1. 학생 목록에서 삭제하려는 학생 옆의 삭제 버튼을 클릭합니다
2. 해당 학생과 관련된 모든 수업도 함께 삭제됩니다

### 2. 시간표 페이지 (`/schedule`)

#### 수업 추가하기 (드래그 앤 드롭)

1. 우측 학생 패널에서 원하는 학생을 찾습니다
2. 학생을 마우스로 드래그하여 원하는 시간대의 셀에 드롭합니다
3. "수업 추가" 모달이 열립니다
4. 과목, 시작 시간, 종료 시간을 선택합니다
5. "수업 추가" 버튼을 클릭합니다

#### 수업 추가하기 (클릭)

1. 원하는 시간대의 빈 셀을 클릭합니다
2. "수업 추가" 모달이 열립니다
3. 학생 이름을 입력하고 Enter 키를 누릅니다
4. 과목, 시작 시간, 종료 시간을 선택합니다
5. "수업 추가" 버튼을 클릭합니다

#### 수업 편집하기

1. 기존 수업 블록을 클릭합니다
2. "수업 편집" 모달이 열립니다
3. 원하는 정보를 수정합니다
4. "수정" 버튼을 클릭합니다

#### 학생별 필터링

1. 우측 학생 패널에서 특정 학생을 클릭합니다
2. 해당 학생의 수업만 시간표에 표시됩니다
3. 다시 클릭하면 모든 학생의 수업이 표시됩니다

#### PDF 다운로드

1. 우측 상단의 "PDF 다운로드" 버튼을 클릭합니다
2. 현재 시간표가 PDF 파일로 다운로드됩니다

### 3. 사용자 매뉴얼 페이지 (`/manual`)

- 현재 페이지에서 애플리케이션의 사용법을 확인할 수 있습니다
- 배포 상태 정보를 확인할 수 있습니다

## ⚙️ 고급 기능

### 시간 검증 시스템

- **시작 시간이 종료 시간보다 늦을 수 없습니다**
- 잘못된 시간을 입력하면 경고 메시지가 표시됩니다
- 시간은 30분 단위로 입력됩니다

### 다중 학생 그룹 수업

- 하나의 수업에 여러 학생을 추가할 수 있습니다
- 학생 이름을 입력하고 Enter 키를 누른 후, 다음 학생 이름을 입력합니다
- 모든 학생이 추가될 때까지 반복합니다

### 겹치는 세션 처리

- 동일한 시간대에 여러 수업이 있을 경우 개별적으로 표시됩니다
- 각 수업의 정보를 명확하게 확인할 수 있습니다

### 입력 개선

- 학생 이름 입력 후 Enter 키를 누르면 입력창이 완전히 초기화됩니다
- 존재하지 않는 학생을 추가하려고 하면 피드백 메시지가 표시됩니다

### 플로팅 패널 관리

- **드래그 가능한 패널**: 수강생 리스트 패널을 마우스로 드래그하여 원하는 위치로 이동할 수 있습니다
- **위치 자동 저장**: 패널 위치가 자동으로 저장되어 다음 방문 시 동일한 위치에 표시됩니다
- **직관적 피드백**: 마우스 커서가 손 모양으로 변경되어 드래그 가능함을 알려줍니다
- **화면 경계 보호**: 패널이 화면 밖으로 나가지 않도록 자동으로 제한됩니다

## 🔧 문제 해결

### 일반적인 문제

#### 데이터가 사라졌어요

- 브라우저의 localStorage가 삭제되었을 수 있습니다
- 다른 브라우저나 시크릿 모드에서는 데이터가 공유되지 않습니다

#### 수업이 표시되지 않아요

- 학생이 선택되어 있는지 확인하세요
- 시간대가 올바른지 확인하세요
- 과목이 선택되어 있는지 확인하세요

#### 모달이 잘려 보여요

- 모달 내용이 많을 경우 스크롤이 가능합니다
- 브라우저 창 크기를 조정해보세요

#### PDF 다운로드가 안 돼요

- 브라우저의 팝업 차단을 해제해주세요
- 인터넷 연결을 확인해주세요

### 지원 브라우저

- Chrome (권장)
- Firefox
- Safari
- Edge

## 💡 팁과 트릭

### 효율적인 사용법

1. **학생 그룹화**: 비슷한 수준의 학생들을 그룹으로 묶어 수업하세요
2. **시간대 활용**: 30분 단위를 활용하여 세밀한 일정 관리가 가능합니다
3. **색상 구분**: 과목별 색상을 활용하여 시각적으로 구분하세요
4. **PDF 백업**: 중요한 일정은 PDF로 다운로드하여 백업하세요

### 단축키

- **Enter**: 학생 추가, 모달에서 확인
- **Escape**: 모달 닫기
- **Tab**: 입력 필드 간 이동
