## 📊 프로젝트 현황

### ✅ 완료된 기능

- 학생 관리 (추가, 삭제, 선택)
- 과목 관리 (추가, 삭제, 편집, 색상 선택)
- 시간표 표시 및 드래그 앤 드롭
- 충돌 해결 로직
- Google OAuth 로그인
- 다크/라이트 테마
- 반응형 디자인
- PDF 다운로드

### 🏗️ 기술 스택

- **Frontend**: Next.js 15, React 18, TypeScript
- **Styling**: TailwindCSS
- **Backend**: Supabase (PostgreSQL + JSONB)
- **Authentication**: Supabase Auth (Google OAuth)
- **Testing**: Vitest, Playwright, React Testing Library
- **Deployment**: Vercel
